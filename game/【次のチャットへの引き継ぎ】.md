# ビッグファイブカードゲーム 引き継ぎメモ
最終更新日：2026年3月21日

---

## プロジェクトの目的
ビッグファイブ理論をベースにした**教育カードゲーム**。印刷して遊べる＋ブラウザで遊べる形で無料公開予定（GitHub Pages）。
将来的には「あそびラボ」ポータルサイトの一つとして公開予定。

## game_demo.html の現在仕様

- **3人対戦**（自分 ＋ CPU α ＋ CPU β）
- **パラメータースケール：10〜50**
- **フィールド：毎回ランダム選択**（13種類から）
- 性別選択 → 難易度選択 → サウンドON/OFF → 診断テスト入力 → ゲーム開始

### カードデッキ（2026-03-21 更新）
- **ATK：20枚**（A-001〜A-022）
- **DEF：24枚**（D-001〜D-025）
- **合計44枚**（旧22枚から倍増）
- 新カード画像は仮置き（既存画像を流用）。要差し替えのカード：
  - ATK：夜更かし・スマホ依存・怠惰先延ばし・暴言悪口・引きこもり・衝動買い・過食・完璧主義の罠（8枚）
  - DEF：ヨガストレッチ・栄養バランスの食事・友人との交流・外国語学習・楽器音楽・自然散歩・整理整頓・資格勉強・新しい趣味・睡眠管理（10枚）

### パラメーター定義

| 略称 | 名称 | 方向性 |
|---|---|---|
| O | 開放性 | 高いほど良い |
| C | 誠実性 | 高いほど良い |
| E | 外向性 | 高いほど良い |
| A | 協調性 | 高いほど良い |
| N | メンタル強さ | 高いほど良い |

### 難易度（2026-03-21 更新・3段階に変更）

| 難易度 | CPU α（index 1・異性別） | CPU β（index 2・同性別） |
|---|---|---|
| 🌱 イージー | ATKカード優先でCPU βを攻撃 | 最適プレイ |
| 🎮 ノーマル | **ランダム**にカード・攻撃対象を選択 | 最適プレイ |
| 💀 ハード | 最適プレイ | 最適プレイ |

- デフォルト：イージー
- `G.difficulty` に `'easy'`/`'normal'`/`'hard'` を保存

### BGM（2026-03-21 実装）

| 状態 | ファイル | 音量 |
|---|---|---|
| 通常 | `BGM/km_BGM作成_〜.mp3` | 0.4 |
| プレイヤーが悪い状態 | `BGM/月のダンジョン (1).mp3` | 0.25 |
| プレイヤーがロールモデル | `BGM/朝.mp3` | 0.2 |

- ゲーム開始でBGM再生、終了で停止
- スタート画面にサウンドON/OFFボタン（デフォルトON）
- ゲーム中タイトルバー右端にミュートボタン（🔊/🔇）
- `updateBGM()` が `applyEffect` 末尾で毎回呼ばれ自動切り替え
- BGM切り替えは `switchBGM(id)` で管理（同じIDなら再起動しない）

### 顔アイコン
- 手札エリアの背面（大）に表示
- `.big-face-img` の `transition` は `filter/transform` のみ（`opacity` を含めない）
  - 理由：`transition:all` にすると画像切り替え時に opacity が累積して透明化するバグが起きる

### 勝利条件
- 現在のステータスに最も近いフィールドと一致 → 即時勝利
- デッキ切れ＋全員手札なし → ロールモデル状態のプレイヤー優先、次に距離最小
- 悪い状態（BAD_STATES）のプレイヤーは勝利から除外

### デッキ残り枚数表示
- カードの**上**に表示
- フォントサイズ：18px

### 主要関数
| 関数名 | 役割 |
|---|---|
| `startGame()` | ゲーム開始・初期化 |
| `selectDifficulty(d)` | 難易度選択（'easy'/'normal'/'hard'） |
| `getClosestField(stats)` | 最も近いフィールドを返す |
| `getFaceInfo(stats, gender, isAttacked)` | 顔画像src＋ラベルを返す |
| `isInBadState(pIdx)` | 悪い状態かどうか判定 |
| `flyCard(fromEl, toEl, card, onLand)` | カードアニメ |
| `applyEffect(card, pIdx, targetPIdx)` | カード効果適用 |
| `checkInstantWin()` | 即時勝利チェック |
| `checkEnd()` | デッキ切れ終了チェック |
| `pickRandomCard(pIdx)` | ノーマルモード：ランダムカード選択 |
| `pickRandomTarget(pIdx)` | ノーマルモード：ランダム攻撃対象 |
| `pickAggressiveCard(pIdx)` | イージーモード：ATK優先 |
| `pickBestCard(pIdx)` | ハードモード：最適プレイ |
| `switchBGM(id)` | BGM切り替え（同IDなら再起動しない） |
| `updateBGM()` | プレイヤー状態に応じてBGM自動切り替え |
| `scaleGame()` | 画面スケール調整 |
